Delete it whenever you're ready and start writing your own.

## How to write a new article

Create a new file inside the `_posts/` folder with this naming pattern:

```
_posts/YYYY-MM-DD-your-title.md
```

At the top of each post, add **front matter** between two `---` lines:

```yaml
---
title: "My great article"
date: 2026-06-14
tags: [webdev, tips]
---
```

Then write your content in **Markdown** below it.

## Markdown basics

You can use **bold**, *italic*, [links](https://github.com), and lists:

- First point
- Second point
- Third point

> Blockquotes look like this — great for highlighting an idea.

Add code blocks with triple backticks:

```js
function greet(name) {
  return `Hello, ${name}!`;
}
```

That's it — commit the file, push to GitHub, and your article goes live.
